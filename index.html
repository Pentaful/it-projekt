Informationen über die Website

Server-Hardware und Betriebssystem

  • Raspberry Pi mit Linux Distribution
  • Rasbian OS auf mikroSD-Karte
  • 64-bit SoC Quad-Core CPU mit 1.8GHz
  • Grafikeinheit mit 500MHz
  • 4GB LPDDR4-3200 RAM
  • WLAN, LAN, Bluetooth und BLE
  • GPIO-Pins

Netzwerkkonfiguration

  • Apache2 als Webserver
  • Modifizierte Apache2 Config erlaubt erheblich niedrigere Latenz (~250ms => ~30ms)
  • Geöffnete Ports: 22 (SSH), 80 (HTTP), 443 (HTTPS), 3000 (Node.js) und 8443
  • Verwendung von DDNS durch DuckDNS
  • Automatische Aktualisierung der IP des Routers durch DDNS, wenn sie sich durch einen Neustart ändert
  • SSL-Zertifikat durch Certbot konfiguriert (=>HTTPS)

Features und Datenbanken

  • MariaDB als Datenbanksystem
  • SQL als Datenbanksprache
  • Mehrere Datensätze zur klaren Trennung von Daten
  • Kommunikation von Webseite zu Datenbank über PHP

Development

  • Visual Studio Code mit SSH-Erweiterung für die Remote-Entwicklung
  • SSH-Verbindung zum Server für das normale Linux-Terminal
